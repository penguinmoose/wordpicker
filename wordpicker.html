instructions for word picker

Welcome to word picker!
You can find words that match a spelling pattern by typing text in the input field using the table provided below. For example, the spelling pattern consonant-vowel-consonant will become CVC.

any lowercase letter = letter
capical C = any consonants
capital V = any vowel
, = or
- = any number of letter
[any character with meaning][any number] = the character before it gets doubled the number of times, but the consecutive characters can be any of the meaning of it, and it is the same
